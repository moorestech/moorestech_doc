# Playerinventory Response Protocol

プレイヤーインベントリを返すプロトコルです。

## 構造

| Type  | 名前                               | デフォルト値 | 備考 | 
| :---: | :--------------------------------: | :----------: | :--: | 
| short | パケットID                          | 4            |      | 
| int   | プレイヤーID                         |              |      | 
|       | パディング                           |              |      | 
| int   | スロット0のアイテムID                 |              |      | 
| int   | スロット0のアイテム数                 |              |      |
| int   | スロット1のアイテムID                 |              |      |
| int   | スロット1のアイテム数                 |              |      |
　以下繰り返し 
| int   | スロット43のアイテムID                |              |      | 
| int   | スロット43のアイテム数                |              |      | 
| int   | スロット44のアイテムID                |              |      | 
| int   | スロット44のアイテム数                |              |      |
| int   | クラフトスロット0のアイテムID          |              |      |
| int   | クラフトスロット0のアイテム数          |              |      | 
| int   | クラフトスロット1のアイテムID          |              |      | 
| int   | クラフトスロット1のアイテム数          |              |      | 
 以下繰り返し
| int   | クラフトスロット8のアイテムID          |              |      |
| int   | クラフトスロット8のアイテム数          |              |      |
| int   | クラフト結果のアイテムID              |              |      |
| int   | クラフト結果のアイテム数              |              |      |
| byte  | クラフト可能か　可能なら1不可能なら0   |              |      |
