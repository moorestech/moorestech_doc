# Blockinventory Response Protocol

ブロックインベントリを返すプロトコルです。

## 構造

| Type  | 名前                               | デフォルト値 | 備考 | 
| :---: | :--------------------------------: | :----------: | :--: | 
| short | パケットID                          | 6            |      | 
| short | アイテム数n                         |              |      | 
| int   | ブロックのID                        |              |      | 
| int   | スロット0のアイテムID                 |              |      | 
| int   | スロット0のアイテム数                 |              |      |
| int   | スロット1のアイテムID                 |              |      |
| int   | スロット1のアイテム数                 |              |      |
 以下繰り返し
| int   | スロットnのアイテム数                 |              |      |
| short | UIタイプID                           |              |      |
