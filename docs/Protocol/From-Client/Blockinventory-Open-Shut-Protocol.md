# Blockinventory-Open-Shut-Protocol

ブロックのインベントリを開いたり閉じたりするプロトコルです。

## 構造

| Type  | 名前                               | デフォルト値 | 備考 | 
| :---: | :--------------------------------: | :----------: | :--: | 
| short | パケットID                          | 16            |      | 
| int   | ブロック座標X                        |              |      | 
| int   | ブロック座標Y                        |              |      | 
| int   | プレイヤーID                         |              |      | 
| byte  | 1なら開く 2なら閉じる                 |              |      |
