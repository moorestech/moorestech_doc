# Player Position Transmission

プレイヤーの現在位置を送信するためのプロトコルです

## 構造

| Type  | 名前                               | デフォルト値 | 備考 | 
| :---: | :--------------------------------: | :----------: | :--: | 
| short | パケットID                         | 2            |      | 
| float | プレイヤーのX座標                   |              |      | 
| float | プレイヤーのy座標                   |              |      | 
| int   | プレイヤーID                       |              |      |
